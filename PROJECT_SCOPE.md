# Project Scope and Disclosure

This repository is prepared for resume and interview use.

## 中文披露

本仓库用于求职作品集展示。这里的项目属于毕设级 / 课程设计级小项目，适合用来说明我做过的前后端开发、接口设计、数据库脚本、测试和文档整理，不应包装成资深工程师参与的大型企业级落地系统。

原始本地 `JXY`、`CWH` 目录中的内容，属于匿名化整理后的同学委托开发 / 协助完成毕业设计项目案例。公开仓库中只保留可讨论的技术实现，不公开姓名、学校材料、论文、答辩文件和运行期隐私数据。

## Honest Positioning

The projects are undergraduate graduation-project-level / course-project-level applications. They demonstrate practical coding, full-stack assembly, documentation, and testing habits, but they should not be treated as enterprise-grade production systems.

## Source Context

The original local source folders were named `JXY` and `CWH`. Before publishing, the content was reorganized and anonymized into English project directories:

- `JXY` -> `projects/warehouse-management-system`
- `CWH` -> `projects/dance-motion-analysis-system`

These cases are described as **classmate-commissioned / collaborative graduation-project development cases**. This phrasing is intentionally used to be honest without exposing private personal details.

## What Was Removed

The following categories are intentionally excluded:

- personal names and school-specific materials
- thesis papers, defense documents, task books, and review drafts
- generated build output such as `target/`, `dist/`, caches, and logs
- virtual environments and dependencies such as `.venv/` and `node_modules/`
- runtime uploads, videos, local JSON records, model binaries, and temporary output
- real credentials, `.env` files, local machine settings, and IDE files

## Intended Use

This repository is suitable as:

- a resume portfolio link
- an interview code sample
- a basis for explaining project structure and implementation tradeoffs

It is not intended as:

- proof of senior enterprise delivery experience
- a deploy-ready SaaS or production product
- a public archive of classmates' original school materials
