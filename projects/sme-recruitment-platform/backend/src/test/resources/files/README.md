# Test File Fixtures / 测试文件夹具

Public repository note: binary upload samples are intentionally not committed.

公开仓库说明：上传样例二进制文件不提交。

Backend upload tests construct small in-memory `MockMultipartFile` instances directly in the test code, so no resume, report evidence, qualification, or private document sample is required in this directory.

后端上传测试直接在测试代码中构造小型内存 `MockMultipartFile`，因此此目录不需要放置简历、举报证据、资质文件或私人文档样例。
