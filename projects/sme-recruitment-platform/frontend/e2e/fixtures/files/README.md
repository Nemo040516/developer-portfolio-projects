# Test File Fixtures / 测试文件夹具

Public repository note: binary upload samples are intentionally not committed.

公开仓库说明：上传样例二进制文件不提交。

Current Playwright tests use `e2e/support/file-fixtures.js` to construct small in-memory PDF/JPG/PNG upload payloads. This keeps E2E coverage available without publishing resume, report evidence, qualification, or private document samples.

当前 Playwright 测试通过 `e2e/support/file-fixtures.js` 构造小型内存 PDF/JPG/PNG 上传内容。这样既保留 E2E 覆盖，又不会公开简历、举报证据、资质文件或私人文档样例。
