# Scripts / 脚本

## 中文说明

这里存放舞蹈动作视频比对与纠错系统的开发辅助脚本，不属于主运行链路。

学校材料、论文、答辩和文档生成相关脚本已从公开作品集仓库中排除。

## English

This directory contains development helper scripts for the dance motion analysis system. They are not part of the main runtime path.

School-specific thesis, defense, and document-generation scripts are intentionally excluded from this public portfolio repository.

## Directories / 子目录

### `dev/`

- `reset_runtime_storage.py`: resets local runtime files for development. / 重置本地运行期文件。
- `generate_nonperfect_analysis_sample.py`: creates a non-perfect sample analysis record for UI review. / 生成非满分分析样例，用于界面复盘。
